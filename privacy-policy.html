Privacy Policy for Netflix IMDb Ratings Extension

Effective Date: 05/01/2025

1. Information We Collect

This extension does not collect, store, or share any personal information. It uses the OMDb API to fetch movie and TV show ratings based on the titles displayed on Netflix.

2. How We Use Information

The data fetched from the OMDb API is:

  • Used only to display IMDb ratings on Netflix thumbnails.
  • Cached locally in the browser’s Chrome Storage API for faster performance and to avoid repeated API calls.

3. Data Security

We use Chrome Storage API, which is sandboxed and isolated by Chrome’s security model. No data is sent to any third-party servers beyond the OMDb API requests required for fetching ratings.

4. Permissions

The extension requires the following permissions:

  • Storage – To cache ratings locally for performance improvements.
  • Unlimited Storage – To handle caching large amounts of data without limits.
  • Host Permissions – To fetch ratings securely from OMDb API.

5. Third-Party Services

This extension communicates only with the OMDb API to fetch publicly available data about movies and TV shows. Refer to OMDb’s Privacy Policy for more details.
